# MLIP-AutoPipe Package

This directory contains the source code for the MLIP-AutoPipe application.

## Structure

*   `config/`: Pydantic schemas for configuration.
*   `data_models/`: Core data structures exchanged between modules.
*   `inference/`: Inference engine components (MD, Embedding, Masking).
*   `core/`: Core utilities (Database, Logging).
*   `dft/`: DFT calculation components.
*   `generator/`: Structure generation components.
*   `surrogate/`: Surrogate model and sampling components.
*   `training/`: Training components.
*   `utils/`: General utilities.

## Usage

This package is intended to be used as a library or via the `mlip-auto` CLI.
